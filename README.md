
# Introduce

Jetpack Compose 기반 UI 아키텍처 학습 프로젝트(당근마켓 스타일의 구조를 참고) - 진행중


## Project Goal
- Jetpack Compose 기반 화면 설계 능력 향상 ✅
- Single Activity + Navigation Compose 구조 이해 ✅
- Feature 단위 패키지 구조 설계 연습 ✅
- 상태 기반 UI 설계 (UiState) ✅
- BottomSheet + Scrim 제어 로직 구현 ✅
- ViewModel을 활용한 화면 상태 관리 ✅
- DI 기반 모듈 구성 연습 ✅
