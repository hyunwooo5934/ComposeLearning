# Android Compose Learning

Jetpack Compose 기반 Android 아키텍처 학습 및 실습 프로젝트.


## Project Goal
- Jetpack Compose UI 구조 이해
- MVVM + Clean Architecture 기반 화면 설계
- Navigation Compose 구조 설계



## Architecture
- MVVM
- Clean Architecture (Presentation / Domain / Data)
- Hilt DI
- Repository Pattern

- Kotlin
- KotlinJetpack Compose
- KotlinMaterial3
- KotlinHilt
- KotlinCoroutine / Flow
